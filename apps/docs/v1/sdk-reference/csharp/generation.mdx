---
title: "Generation lookup"
description: "Call /generation to retrieve a past generation."
---

**Method**: Direct HTTP GET /generation/{id}

### Example
```csharp
var client = new HttpClient();
client.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", apiKey);

var response = await client.GetAsync("https://api.ai-stats.phaseo.app/v1/generation/G-01ABC...");
```

### Parameters
- `id` (string, required): Request id returned by a previous call.

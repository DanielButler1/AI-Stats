---
title: "Models"
description: "List models with the C# SDK."
---

**Method**: `client.GetModels()`

### Example
```csharp
var client = new Client(apiKey);

var models = client.GetModels();
```

### Parameters
- Optional filters: `provider`, `organisation`, `limit`, `offset`.

### Returns

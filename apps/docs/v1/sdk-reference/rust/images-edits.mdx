---
title: "Image edits"
description: "Call /images/edits with the Rust SDK."
---

**Method**: `images_api::images_edits_post()`.

### Example
```rust
use ai_stats_sdk::apis::{configuration::Configuration, images_api};
use ai_stats_sdk::models::ImagesEditRequest;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut config = Configuration::new();
    config.bearer_access_token = Some("your-api-key".to_string());

    let request = ImagesEditRequest::new(
        "openai/dall-e-2".to_string(),
        "A cat sitting on a chair".to_string(),
        "base64-encoded-image".to_string()
    );

    let response = images_api::images_edits_post(&config, request).await?;
    println!("Edited image: {:?}", response);
    Ok(())
}
```

### Key parameters
- `model` (required): The model to use for image editing.
- `prompt` (required): Description of the desired edit.
- `image` (required): Base64 encoded image to edit.
- `mask` (optional): Base64 encoded mask.
- `n` (optional): Number of images to generate.
- `size` (optional): Size of the output image.
- `user` (optional): End-user identifier.

### Returns
`ImageGenerationResponse` containing the edited images.

---
title: "Moderations"
description: "Call /moderations with the Rust SDK."
---

**Method**: `moderations_api::moderations_post()`.

### Example
```rust
use ai_stats_sdk::apis::{configuration::Configuration, moderations_api};
use ai_stats_sdk::models::{ModerationRequest, ModerationRequestInput};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut config = Configuration::new();
    config.bearer_access_token = Some("your-api-key".to_string());

    let request = ModerationRequest::new(
        "openai/omni-moderation-latest".to_string(),
        ModerationRequestInput::String("Text to classify".to_string())
    );

    let response = moderations_api::moderations_post(&config, request).await?;
    println!("Moderation result: {:?}", response);
    Ok(())
}
```

### Key parameters
- `model` (required): The model to use for moderation.
- `input` (required): Text or array of content items to classify.

### Returns
`ModerationResponse` containing moderation results.

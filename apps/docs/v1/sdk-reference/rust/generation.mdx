---
title: "Generation lookup"
description: "Call /generation with the Rust SDK."
---

**Method**: `generations_api::generation_get()`.

### Example
```rust
use ai_stats_sdk::apis::{configuration::Configuration, generations_api};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut config = Configuration::new();
    config.bearer_access_token = Some("your-api-key".to_string());

    let record = generations_api::generation_get(&config, "request-id").await?;
    println!("Generation record: {:?}", record);
    Ok(())
}
```

### Key parameters
- `id` (required): The request ID of the generation to retrieve.

### Returns

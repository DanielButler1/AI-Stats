---
title: "Health"
description: "Call /healthz with the Rust SDK."
---

**Method**: `analytics_api::healthz_get()`.

### Example
```rust
use ai_stats_sdk::apis::{configuration::Configuration, analytics_api};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut config = Configuration::new();
    config.bearer_access_token = Some("your-api-key".to_string());

    let health = analytics_api::healthz_get(&config).await?;
    println!("Health status: {:?}", health);
    Ok(())
}
```

### Key parameters
None.

### Returns
`HealthzGet200Response` containing health information.

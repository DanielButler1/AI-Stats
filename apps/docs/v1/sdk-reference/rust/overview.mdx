description: "Generated client for the AI Stats Gateway. Models endpoint only for now."

- Status: Preview (not yet on crates.io).
- Generate with `pnpm openapi:gen:rust`.
- Wrapper `packages/sdk-rust/src/lib.rs` exposes `get_models`.

```rust
let client = ai_stats_sdk_example::Client::new("YOUR_KEY", "https://api.ai-stats.phaseo.app/v1");
let resp = client.get_models(None, Some(5), None).await?;
```

TypeScript and Python SDKs are production-ready; other languages follow soon.

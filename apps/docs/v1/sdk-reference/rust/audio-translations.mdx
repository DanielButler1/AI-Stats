---
title: "Audio translations"
description: "Call /audio/translations with the Rust SDK."
---

**Method**: `audio_api::audio_translations_post()`.

### Example
```rust
use ai_stats_sdk::apis::{configuration::Configuration, audio_api};
use ai_stats_sdk::models::AudioTranslationRequest;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut config = Configuration::new();
    config.bearer_access_token = Some("your-api-key".to_string());

    let mut request = AudioTranslationRequest::new("openai/whisper-1".to_string());
    request.audio_url = Some("https://example.com/audio.mp3".to_string());

    let response = audio_api::audio_translations_post(&config, request).await?;
    println!("Translation: {}", response.text.unwrap_or_default());
    Ok(())
}
```

### Key parameters
- `model` (required): The model to use for translation.
- `audio_url` (optional): URL of the audio file to translate.
- `audio_b64` (optional): Base64 encoded audio data.
- `language` (optional): Target language for translation.
- `prompt` (optional): Prompt to guide the translation.
- `temperature` (optional): Sampling temperature.

### Returns
`AudioTranscriptionResponse` containing the translated text.

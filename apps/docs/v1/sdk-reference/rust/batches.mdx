---
title: "Batches"
description: "Call /batches with the Rust SDK."
---

**Method**: `batch_api::batches_post()`.

### Example
```rust
use ai_stats_sdk::apis::{configuration::Configuration, batch_api};
use ai_stats_sdk::models::BatchRequest;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut config = Configuration::new();
    config.bearer_access_token = Some("your-api-key".to_string());

    let request = BatchRequest::new(
        "file-abc123".to_string(),
        "/v1/chat/completions".to_string()
    );

    let response = batch_api::batches_post(&config, request).await?;
    println!("Batch ID: {}", response.id.unwrap_or_default());
    Ok(())
}
```

### Key parameters
- `input_file_id` (required): The ID of the input file.
- `endpoint` (required): The API endpoint to call.
- `completion_window` (optional): Completion window for the batch.
- `metadata` (optional): Additional metadata.

### Returns
`BatchResponse` containing batch details and status.

---
title: "Models"
description: "Call /models with the Rust SDK."
---

**Method**: `client.get_models()`.

### Example
```rust
use ai_stats_sdk::Client;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Client::new("your-api-key", "https://api.ai-stats.phaseo.app/v1");

    let models = client.get_models(None, None, None).await?;
    println!("{:?}", models);
    Ok(())
}
```

### Key parameters
- `provider` (optional): Filter by provider.
- `limit` (optional): Maximum number of models to return.
- `offset` (optional): Number of models to skip.

### Returns
`ModelListResponse` containing a list of available models.

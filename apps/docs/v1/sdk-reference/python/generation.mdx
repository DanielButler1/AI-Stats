---
title: "Generation lookup"
description: "Call /generation to retrieve a past generation."
---

**Method**: `client.get_generation()`.

### Example
```python
record = client.get_generation("G-01ABC...")
```

### Parameters
- `generation_id` (string, required): Request id returned by a previous call.

**Method**: `AIStatsImagesApi::imagesEditsPost()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsImagesApi.h"
#include "ai_stats_sdk_cpp/model/AIStatsImagesEditRequest.h"

// Setup config and apiClient as in usage.mdx
AIStatsImagesApi imagesApi(apiClient);

auto request = std::make_shared<AIStatsImagesEditRequest>();
request->setModel("openai/gpt-image-1");
request->setPrompt("Make it sunset");
request->setImage("data:image/png;base64,...");

auto response = imagesApi.imagesEditsPost(request).get();
```

### Key parameters
- `model` (required): Image-edit capable model id.
- `image` (required): Base64 data URL or URL to source image.
- `prompt`: Optional new instruction for the edit.
- `mask`: Optional mask (transparent regions edited).
- `n` (1â€“10): Number of edited variants.
- `size`: e.g., `1024x1024`, `1024x1792`, `1792x1024`.
- `user`: Optional end-user tag.

### Returns
`AIStatsImageGenerationResponse`

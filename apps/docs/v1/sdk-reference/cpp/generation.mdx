---
title: "Generation lookup"
description: "Call /generation to retrieve a past generation."
---

**Method**: `AIStatsGenerationsApi::generationGet()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsGenerationsApi.h"

// Setup config and apiClient as in usage.mdx
AIStatsGenerationsApi generationsApi(apiClient);

auto record = generationsApi.generationGet("G-01ABC...").get();
```

### Parameters
- `id` (string, required): Request id returned by a previous call.

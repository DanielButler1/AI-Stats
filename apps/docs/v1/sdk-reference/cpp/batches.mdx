title: "Batches"
description: "Call /batches with the C++ SDK."
---

**Methods**: `AIStatsBatchApi::batchesPost()`, `AIStatsBatchApi::batchesBatchIdGet()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsBatchApi.h"
#include "ai_stats_sdk_cpp/model/AIStatsBatchRequest.h"

// Setup config and apiClient as in usage.mdx
AIStatsBatchApi batchApi(apiClient);

auto request = std::make_shared<AIStatsBatchRequest>();
request->setEndpoint("responses");
request->setInputFileId("file_123");
request->setCompletionWindow("24h");

auto batch = batchApi.batchesPost(request).get();
auto status = batchApi.batchesBatchIdGet(batch->getId()).get();
```

### Key parameters
- `endpoint` (required): Target endpoint for batch items (e.g., `responses`).
- `input_file_id` (required): File id uploaded via `/files`.
- `completion_window`: e.g., `24h`.
- `metadata`: Optional object stored with the batch.

### Returns
`AIStatsBatchResponse`

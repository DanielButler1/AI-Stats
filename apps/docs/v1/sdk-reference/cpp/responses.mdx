---

**Method**: `AIStatsResponsesApi::createResponse()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsResponsesApi.h"
#include "ai_stats_sdk_cpp/model/AIStatsResponsesRequest.h"

// Setup config and apiClient as in usage.mdx
AIStatsResponsesApi responsesApi(apiClient);

auto request = std::make_shared<AIStatsResponsesRequest>();
request->setModel("openai/gpt-4.1");
request->setInput({utility::conversions::to_string_t(R"([{"role":"user","content":[{"type":"output_text","text":"Summarise this text"}]}])")});
request->setTemperature(0.7);

auto response = responsesApi.createResponse(request).get();
```

### Key parameters
- `model` (required): Target model id.
- `input` (required): Ordered array of input items (messages, tool calls, etc.).
- `temperature` (0–2): Higher = more random.
- `top_p` (0–1) / `top_k` (>=1): Nucleus / k-best sampling controls.
- `max_output_tokens` (int): Hard cap on tokens generated per response.

### Returns
`AIStatsResponsesResponse`

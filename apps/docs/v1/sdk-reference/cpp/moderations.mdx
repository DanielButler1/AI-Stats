**Method**: `AIStatsModerationsApi::moderationsPost()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsModerationsApi.h"
#include "ai_stats_sdk_cpp/model/AIStatsModerationRequest.h"

// Setup config and apiClient as in usage.mdx
AIStatsModerationsApi moderationsApi(apiClient);

auto request = std::make_shared<AIStatsModerationRequest>();
request->setModel("openai/omni-moderation-latest");
request->setInput("Text to classify");

auto response = moderationsApi.moderationsPost(request).get();
```

### Key parameters
- `model` (required): Moderation model id (e.g., `openai/omni-moderation-latest`).
- `input` (required): String or array of content items (text or image_url).

### Returns
`AIStatsModerationResponse`

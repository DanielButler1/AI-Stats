---
title: "Models"
description: "List models with the C++ SDK."
---

**Method**: `AIStatsModelsApi::modelsGet()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsModelsApi.h"

// Setup config and apiClient as in usage.mdx
AIStatsModelsApi modelsApi(apiClient);

auto models = modelsApi.modelsGet().get();
```

### Parameters
- Optional filters: `provider`, `organisation`, `endpoints[]`, `limit`, `offset`.

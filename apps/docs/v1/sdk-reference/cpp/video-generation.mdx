---

**Method**: `AIStatsVideoApi::videoGenerationPost()`.

### Example
```cpp
#include "ai_stats_sdk_cpp/ApiClient.h"
#include "ai_stats_sdk_cpp/ApiConfiguration.h"
#include "ai_stats_sdk_cpp/api/AIStatsVideoApi.h"
#include "ai_stats_sdk_cpp/model/AIStatsVideoGenerationRequest.h"

// Setup config and apiClient as in usage.mdx
AIStatsVideoApi videoApi(apiClient);

auto request = std::make_shared<AIStatsVideoGenerationRequest>();
request->setModel("openai/gpt-video-1");
request->setPrompt("A calm ocean at dusk");

auto response = videoApi.videoGenerationPost(request).get();
```

### Key parameters
- `model` (required): Video-capable model id.
- `prompt` (required): Text description of the video.

### Returns
`AIStatsVideoGenerationResponse`
